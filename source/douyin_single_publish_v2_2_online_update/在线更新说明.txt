抖音单图文发布 v2.2.0 在线更新版

更新功能说明：
1. 程序启动后会自动检查 GitHub 上的 version.json。
2. 前端增加“检查更新”按钮，可手动检查。
3. 检测到新版本后会弹窗询问是否更新，用户可自行选择。
4. 更新时会下载 packages/douyin_single_publish_v*.zip，校验 sha256 后覆盖程序文件。
5. 本机配置 app/douyin_gui_config.json 会保留，不会被新版包覆盖。

后续发新版本的方法：
1. 修改程序代码后，把整个程序目录重新压缩成 packages/douyin_single_publish_v新版本.zip。
2. 计算这个压缩包的 sha256。
3. 修改仓库根目录 version.json：latest_version、download_url、sha256、release_notes。
4. 提交到 GitHub main 分支。
5. 用户打开旧版本后会自动提示更新。
