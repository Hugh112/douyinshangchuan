抖音单图文发布v2.1 使用说明

本版是 Python + Playwright/CDP 后台稳定版。

新增/保留功能：
1. 浏览器仍然可以后台运行，脚本通过 CDP 控制页面。
2. 不需要你手动修改现有浏览器快捷方式目标；前端会临时追加 remote-debugging-port 和 user-data-dir 参数，不会改写 .lnk 文件本身。
3. 新增多日定时：前端填写“开始日期”和“结束日期”，一次运行可排多天。
4. 保留 v1.1.1 的发布页重试、文案填入后标记已用、暂停弹窗、单标签、热门榜音乐、发布后删除图片/文案等逻辑。

前端新增项：
- 浏览器用户目录(可空)：
  可填写 D:\douyin_chrome_zhejiang。留空时，如果快捷方式没有 user-data-dir，脚本会临时使用当前 Windows 用户目录下的 douyin_chrome_profile_v21。
- 浏览器Profile目录(可空)：
  需要指定 Profile 20 时填写 Profile 20；普通独立目录建议留空。
- 复用已存在调试端口：
  勾选时，如果 9222 已存在就直接复用；如果经常复用到错误浏览器，可取消勾选或勾选“启动前关闭Chrome残留”。
- 启动前关闭Chrome残留：
  会 taskkill chrome.exe 和 node.exe，适合专用电脑/专用账号；如果你同时开普通 Chrome，不建议勾选。
- 后台运行不拉起浏览器：
  默认勾选，脚本不会主动置顶浏览器。

多日定时示例：
开始日期：2026-06-25
结束日期：2026-06-27
开始小时：8
结束小时：18
分钟点：5,25,45
每小时条数：3
会生成 2026-06-25、2026-06-26、2026-06-27 三天的发布时间。

启动：
Start_Douyin_Publisher_v2.1.vbs

诊断启动：
app\debug_start.bat

依赖：
如果新电脑缺依赖，可运行：安装依赖_可选.bat
